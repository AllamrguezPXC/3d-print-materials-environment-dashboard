# Evidence Capture Skill

Use this skill to maintain assignment evidence.

## Evidence checklist

- Plan Mode + Ask Mode
- `/init` and `CLAUDE.md`
- TDD cycle
- Documentation generation
- Security review
- GitHub MCP action
- Custom Skill
- Custom Hook

## Rules

1. Be honest about what is complete and incomplete.
2. Prefer concise evidence with dates and file references.
3. Do not fake screenshots or logs.
4. Link to files under `/evidence` when possible.
5. Keep `EVIDENCE.md` aligned with actual repository content.
